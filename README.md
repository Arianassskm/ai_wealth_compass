## 开发环境设置

1. **前端开发**
```bash
cd frontend
npm install
npm run dev
```

2. **后端开发**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

3. **数据库设置**
```bash
createdb ai_wealth_management
cd backend
alembic upgrade head
```

4. **Redis设置**
```bash
# 安装Redis
# 启动Redis服务
redis-server

# 测试Redis连接
redis-cli ping
```

5. **环境变量配置**
```bash
# 创建.env文件
cp .env.example .env
# 编辑.env文件设置必要的环境变量
```

- feat: 新功能
